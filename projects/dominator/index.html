I created a program that produces Elements depending on what is inserted.

TagName is what you are wanting to insert, so for example like H1, or div. className is any styling you are wanting to insert as long as it is DOM.

id is classifying the element and innerHTML is if you would like any words to appear.

Parent id is if you are trying to insert something into the previous element. For example, inserting words into a div that was previously created.


The three buttons at the bottom which include: Create New Element, Remove Last Element and Reset, all have specfic functions.

Create New Element inserts the information you have filled out. Remove last Element removes the most recent element that you have created. Reset clears the entire field.
